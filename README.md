# INTERFAZ-EXTERNA

Aplicación de ejemplo con interfaz gráfica en Tkinter. La pantalla principal
permite elegir entre dos opciones:

1. **Ver Base de Datos**: abre una tabla editable con un estilo similar a Excel
   que sirve como base para mostrar datos reales en futuras actualizaciones.
2. **Acciones**: ventana de marcador de posición para futuras funcionalidades.

## Requisitos

- Python 3

## Uso

```bash
python main.py
```

La aplicación se ejecuta en una ventana gráfica; asegúrate de tener un entorno
con soporte para Tk (por ejemplo, un sistema con servidor gráfico).
